.. _explore:

################
Data Exploration
################

The Chronograf Explore tool is the quickest way to query data in Sasquatch.

This tool provides an intuitive interface for building and executing queries for quickly analyzing data without needing to write complex queries manually.

See the :ref:`observatory-telemetry` and :ref:`pipeline-metrics` guides for an overview of the databases available in Sasquatch.

Check the `InfluxQL documentation`_ and `Flux documentation`_ for more information on the query languages supported in Chronograf.

Read more about the `Chronograf Explore`_ tool in the Chronograf documentation.
